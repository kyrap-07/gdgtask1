# Kanban Task Manager

This is a single page Kanban board application developed for GDG on Campus SRM recruitment.

## Features

- Add tasks
- Brainstorm ideas
- Plan tasks
- Drag and drop tasks across columns
- Remove tasks
- Local storage persistence
- Responsive design

## Workflow Stages

- To Do
- Up Next
- In Progress
- In Review
- Complete
